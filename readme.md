# Bententrade — сайт

Многостраничный сайт мебельного бренда **Bententrade (BTT)**, выполненный по референсу CAIRIS: тёмный фон + медно-оранжевый акцент, крупная гротеск-типографика, мягкие скруглённые карточки.

## Страницы
- `index.html` — Главная (статичный герой по референсу, промо, коллекция, материалы)
- `account.html` — Личный кабинет (профиль, история заказов, избранное, адреса, настройки)
- `catalog.html` — Каталог (фильтры по категориям, 11 товаров)
- `about.html` — О нас (история, принципы, статистика)
- `contacts.html` — Контакты (форма + контактные карточки)

## Языки
Переключатель **RU / UZ / EN** в шапке. Тексты — в `assets/i18n.js`, выбор сохраняется в localStorage.

## Файлы
- `assets/styles.css` — токены, типографика, компоненты
- `assets/pages.css` — раскладки страниц
- `assets/i18n.js` — словарь переводов
- `assets/site.js` — переключение языка, корзина, фильтры, форма
- `assets/btt-logo.png` — логотип

## Бренд
- Шрифты: Hanken Grotesque (заголовки/текст), Cormorant Garamond (курсивные надписи под лого)
- Цвета: тёмный `#1b1916`, медный `#bd7335`, акцент `#e27c3d`, крем `#f4f1ea`
- Изображения интерьеров — заглушки с Unsplash, замените на реальные фото товаров.
